# X-Unbookmarker

X-Unbookmarker is a Chrome Extension designed to help you manage your digital footprint on X (formerly Twitter) by automating the process of removing bookmarks.

This tool allows you to bulk-remove bookmarks with a customizable delay and real-time progress tracking, running entirely locally in your browser.

## Download

You can download the latest version of the extension from the Releases page:

[**Download Latest Release**](https://github.com/ahmed-abdelwahed1/X-Unbookmarker/releases/latest)

## Installation

This extension is installed manually via Chrome's Developer Mode.

1.  Download the latest zip file from the link above.
2.  Extract the zip file to a folder on your computer.
3.  Open Google Chrome and navigate to `chrome://extensions/`.
4.  Enable **Developer mode** in the top right corner.
5.  Click **Load unpacked** in the top left corner.
6.  Select the folder where you extracted the extension.

## Features

-   **Automated Cleaning**: automatically finds and removes bookmarks.
-   **Smart Scrolling**: loads more tweets as the current batch is cleared.
-   **Customizable Delay**: adjust the speed to avoid rate limits.
-   **Error Handling**: automatically retries on failure.
-   **Local Processing**: no data is sent to external servers.

## Usage

1.  Log in to your X (Twitter) account.
2.  Navigate to your [Bookmarks page](https://x.com/i/bookmarks).
3.  Click the X-Unbookmarker icon in your browser toolbar.
4.  Click **Start Cleaning**.
5.  (Optional) Adjust the delay slider to control the speed.
